# HospitalWebSite

A Hospital website, a college project.

## Dependencies

+ node & npm
+ mysql or mariadb

## How to use

1. Install the dependencies (node modules):

``` bash
npm install
```

2. Setup the environment with this script, or do it manually. 

``` bash
sudo bash setup.sh
```

3. Run the development script
   
```bash
npm run dev 
```
